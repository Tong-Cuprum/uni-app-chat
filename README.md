前端线上面试题
一、测试目的
考察候选人的UniApp跨平台开发能力、UI设计实现能力、动画交互能力及前端工程化水平。
二、测试内容
借助AI工具例如cursor，开发一个跨平台对话类应用，包含以下四个页面：
页面
功能要求
对话列表
展示会话记录（头像、昵称、最后消息、时间、未读数），支持下拉刷新和上拉加载，点击跳转详情
聊天详情
展示完整对话记录，支持文本和图片消息，左右分布区分发送方，底部输入框支持发送消息，发送后自动滚动并有动画效果
联系人详情
展示联系人完整信息（头像、昵称、个性签名、等级等），支持返回交互
个人中心
展示当前用户信息，提供功能入口展示
数据要求：全部使用Mock数据，数据格式JSON，需足够丰富以支撑完整演示。
三、技术要求
框架：UniApp（Vue3）
平台：Android或iOS真机运行
结构：组件化开发，代码规范
动画：页面转场平滑、消息发送有动画、交互有反馈
加分项：良好的项目结构、状态管理、性能优化、详细注释。
四、评分标准
项目
分值
评分要点
页面美观度
35分
视觉统一、细节精致、适配良好
动画过渡
25分
转场自然、动画流畅、反馈及时

25分
功能可用、数据充分、流程顺畅
代码质量
15分
结构清晰、规范良好、优化合理
